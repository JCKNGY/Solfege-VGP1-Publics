# Solfege

Solfege is a roguelike where music drives combat.
You play an aspiring composer trapped in a zombie apocalypse.
You fight by composing instruments, casting sheet music spells, and staying on beat.

---

## Core Idea

Combat uses rhythm timing.
Your input timing controls damage, defense, and spell strength.
Missing the beat lowers effectiveness.
Staying on time increases output and control.

---

## Features

* Rhythm based combat system
* Summon instruments that fight alongside you
* Cast spells using sheet music patterns
* Roguelike progression with repeat runs
* Difficulty increases over time
* Timing based skill system
* Metronome Beat System

---

## Gameplay Loop

1. Start a run
2. Collect sheet music and upgrades
3. Build your composition setup
4. Fight waves of enemies
5. Improve timing and survive longer
